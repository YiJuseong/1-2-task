Python Console Quiz Game

본 프로젝트는 Python을 활용하여 개발된 콘솔 기반 학습용 퀴즈 프로그램입니다. 사용자가 직접 퀴즈를 등록하고 풀이하며, 학습 데이터를 JSON 파일로 영구 저장하여 관리할 수 있도록 설계되었습니다.

# 1. 프로젝트 개요
개발 언어: Python 3.10+

주요 기능: 퀴즈 풀기, 퀴즈 등록, 목록 조회, 최고 점수 기록, 데이터 영구 저장(JSON)

특징: 클래스 기반의 구조화된 설계, 예외 처리를 통한 사용자 입력 안정성 확보, Git 브랜치 전략 활용 개발.

# 2. 퀴즈 주제 선정 이유
주제: 파일 디렉토리 관련 이동, 삭제 등 기본명령어와 도커 및 깃 관련 기본 명령어
1주차 학습 내용들을 오래 기억할 수 있도록 복습한다는 생각으로 1주차 학습내용을 문제로 선정했습니다.

# 3. 실행 방법
프로젝트 루트 디렉토리에서 아래 명령어를 실행합니다. 별도의 외부 라이브러리 설치가 필요 없는 표준 라이브러리 기반 프로그램입니다.
```bash
python main.py
```
# 4. 기능 상세 목록
퀴즈 풀기: 저장된 퀴즈를 출제합니다. 정답/오답 실시간 판별 및 최종 점수를 합산하여 최고 기록을 갱신합니다.

퀴즈 등록: 새로운 문제, 4개의 선택지, 정답 번호를 입력받아 시스템에 추가합니다. (공백 처리 및 숫자 범위 검증 포함)

목록 조회: 현재 등록된 모든 퀴즈의 제목을 번호와 함께 출력합니다.

최고 점수 확인: 역대 가장 많이 맞힌 정답 개수를 확인합니다.

데이터 유지: 프로그램 종료 시 모든 데이터가 자동으로 저장되며, 재실행 시 기존 데이터를 불러옵니다.

# 5. 파일 구조 및 데이터 설명
## 1. 📂 파일 구성

main.py: Quiz 클래스와 QuizGame 클래스가 포함된 메인 프로그램 소스 코드.

Quiz 클래스 (데이터 담당): 개별 퀴즈의 속성(문제, 보기, 정답)을 관리하고, 자기 자신의 데이터를 출력하거나 정답 여부를 판별하는 원자적 기능에 집중합니다.

QuizGame 클래스 (제어 담당): 게임의 전체 생명주기(Life Cycle)를 관리합니다. 파일 입출력, 메뉴 루프, 점수 합산 등 시스템 전체의 흐름을 제어합니다.

state.json: 퀴즈 데이터와 최고 점수가 저장되는 UTF-8 인코딩 데이터 파일.

.gitignore: state.json 및 캐시 파일의 Git 추적 제외 설정.

README.md: 프로젝트 상세 설명 문서.

## 2.💾 데이터 스키마 (state.json)
데이터는 아래와 같은 JSON 구조로 관리됩니다.

{
    "quizzes": [
        {
            "question": "문제 내용이 들어갑니다.",
            "choices": ["보기1", "보기2", "보기3", "보기4"],
            "answer": 1
        }
    ],
    "best_score": 5
}
state.json 읽기/쓰기 흐름
1. Load: __init__ 시점에 호출. 파일이 있으면 json.load 후 Quiz 객체 리스트로 변환. 없으면 기본 리스트 생성.
2. Save: add_quiz 또는 play_game에서 점수 갱신 시 호출. 객체 리스트를 dict로 변환(to_dict) 후 json.dump로 물리 파일 기록.

# 6. 예외 처리 기준
숫자 입력: 앞뒤 공백 제거, 문자 입력 시 재입력 유도, 허용 범위(1~5 등) 외 숫자 필터링.

비정상 종료: Ctrl+C 또는 EOFError 발생 시 데이터를 안전하게 저장한 후 "비정상 종료" 메시지 없이 종료되도록 처리.

파일 무결성: state.json 파일이 없으면 기본 퀴즈로 시작하며, 파일 손상 시 안내 메시지 후 초기화하여 프로그램 실행 유지.
